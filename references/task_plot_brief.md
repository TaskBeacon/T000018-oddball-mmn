# Task Plot Brief

## Task

- Title: Oddball Task (MMN/P3)
- Construct: novelty processing / target detection
- Paradigm: three-stimulus visual oddball task with standard, deviant, and target conditions.

## Rows

- Standard (frequent non-target, symbol: ○)
- Deviant (infrequent non-target, symbol: △)
- Target (infrequent target, symbol: ★)

## Trial Timeline

1. Fixation: 300 ms, no response.
2. Stimulus response window: 500 ms. Present the condition-specific visual symbol. Press space only for target; withhold response for standard and deviant.
3. Outcome classification happens internally after the response window: hit, miss, false alarm, or correct rejection. There is no per-trial feedback screen.
4. ITI: 500 ms fixation, no response.

## Notes

- Human timing uses 300 ms fixation, 500 ms stimulus, and 500 ms ITI.
- Condition weights are standard 0.70, deviant 0.20, target 0.10.
- The visual should not show an inherited MID cue/anticipation/feedback structure.
