# Task Plot Prompt

Create a clean scientific task-flow timeline on a white background with a blank top header band reserved for later overlay. Do not draw any title, subtitle, logo, watermark, people, device, lab room, or decorative scene.

Use three horizontal rows:

1. Standard
2. Deviant
3. Target

For every row, draw the same left-to-right sequence with arrows:

1. Fixation
   - Screen: fixation cross.
   - Timing: 300 ms.
   - Response label: no response.
2. Stimulus / response window
   - Screen for Standard row: large circle symbol ○.
   - Screen for Deviant row: large triangle symbol △.
   - Screen for Target row: large star symbol ★.
   - Timing: 500 ms.
   - Response label for Standard and Deviant: withhold response.
   - Response label for Target: press SPACE.
3. Outcome classification
   - Standard and Deviant rows: correct rejection or false alarm.
   - Target row: hit or miss.
   - No timing label; mark as internal classification.
   - Make clear this is not a feedback screen.
4. ITI
   - Screen: fixation cross.
   - Timing: 500 ms.
   - Response label: no response.

Add small row notes: Standard 70%, Deviant 20%, Target 10%. Keep labels readable, row spacing consistent, and the timeline content below the reserved header band.
